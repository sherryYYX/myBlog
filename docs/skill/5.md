## 秒的格式化
功能： 将秒数给格式化为 xx 天 xx 小时 xx 分 xx 秒
::: details 代码示例
```javaScript
  /**
 * 格式化秒
 * @param result
 * @returns {string}
 */
   function formatSecond(result) {
    const d = Math.floor((result / 86400) % 24);
    const h = Math.floor((result / 3600) % 24);
    const m = Math.floor((result / 60) % 60);
    const s = Math.floor(result % 60);
    result = s + "秒";
    if (m > 0) {
      result = m + "分" + result;
    }
    if (h > 0) {
      result = h + "小时" + result;
    }
    if (d > 0) {
      result = d + "天" + result;
    }
    return result;
  }

```
